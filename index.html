Features

Everything a background utility should be

Fast because it talks to Windows directly. Quiet because it's supposed to be.

Zero-lag volume

Moves in precise 2% steps through the Windows Audio Session API — the same layer your system mixer uses. No polling, no delay.

Hotkeys that don't care what's focused

Global shortcuts fire even in full-screen games or your IDE. Rebind anything in seconds, right from the app.

Floating mini player

Drag it anywhere on screen. Album art loads at full 600×600 resolution from iTunes.

No Premium needed

No account linking, no OAuth. Free accounts work exactly like Premium ones.

Zero telemetry

Nothing leaves your machine except album art lookups and update checks.

Auto-updates itself

New versions ship straight from GitHub Releases. No separate installer runs.

How it works

Built on Windows, not around it

Other tools scrape window titles or simulate keypresses blindly. KeyTunes talks to the same system APIs Windows itself uses.
